# rtc-verilog
RTC using Verilog

# Overview

This project implements a Real Time Clock (RTC) using Verilog HDL. The design uses counter-based timing logic to maintain seconds, minutes, and hour values.

# Features
1. Seconds, minutes, and hours counter
2. Counter rollover handling
3. Synchronous reset support
4. Simulation testbench included

# Tools Used
1. Verilog HDL
2. Xilinx Vivado

# Design Architecture

The RTC is implemented using:

1. Frequency divider
2. Counter modules
3. Comparator logic
4. Sequential timing logic

# Simulation

The project was verified through waveform simulation.

Simulation validates:

1. Time incrementation
2. Counter rollover
